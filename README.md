# TOFUpaper
Scripts used in the TOFU-MAaPO paper

## Repository Overview

- `assembly_benchmarks/`: Scripts and result files for benchmarking the MAG assembly of TOFU-MAaPO against ATLAS, metaFun, and nf-core/mag on `SRP102150` samples.
- `curMetagenome_processing/`: Batch scripts and analysis code for large-scale curated metagenome sylph profiling.
- `metaphlan_humann_results_SRP102150/`: MetaPhlAn/HUMAnN abundance tables, metadata, and analysis scripts for the `SRP102150` dataset.

### Top-level files

- `curatedmetagenomics_sample_list_sex_age.csv`: Sample metadata table used for large-scale curated metagenome sylph profiling.
